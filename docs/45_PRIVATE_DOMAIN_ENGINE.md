# 45 Private Domain Engine

## Goal

Private Domain Engine prepares community operations for store groups, brand groups, hiking groups, camping groups, VIP groups and event groups.

## Model

`private_domain_groups`

- Group name
- Platform
- Store
- Owner employee
- Customer count
- Group type
- Topic
- Status

## Enterprise WeChat

Enterprise WeChat remains a placeholder until credentials are configured. The system must not send messages automatically without explicit approval.
