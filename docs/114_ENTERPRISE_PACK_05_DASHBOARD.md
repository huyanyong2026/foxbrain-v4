# Enterprise Pack 05 - Unified Dashboard Framework

## Purpose

Pack 05 builds a unified dashboard framework for executive and operational cockpits.

All dashboard data should come from one data service. AI recommendations and alerts must be independent components so they do not pollute raw business data.

## Design Rules

- KPI data comes from the unified dashboard data service.
- Alerts are generated by an independent alert service.
- AI recommendations are generated by an independent recommendation service.
- Business recommendations must display their basis for manager review.
- High-risk actions from recommendations still require human approval.

## Dashboard Types

CEO cockpit:

- Sales
- Gross profit
- Cash flow summary
- Inventory alerts
- Store ranking
- Pending approvals
- AI recommendations
- SAP sync health

Finance dashboard:

- Revenue
- Margin
- Receivables
- Payables
- Cash trend
- Exception alerts

Store dashboard:

- Daily sales
- Conversion
- Inventory health
- Staff performance
- Customer activity
- Suggested replenishment

## Alert Engine

Alert types:

- Low inventory
- Abnormal margin
- Sync failures
- Expiring contracts
- High-value inactive customers

Alert outputs include:

- Alert ID
- Type
- Level
- Message
- Evidence
- Approval requirement

## AI Recommendation Component

Recommendations must include:

- Recommendation ID
- Title
- Basis
- Risk level
- Approval requirement
- Review note

The component is separated from KPI data. Managers should review the basis before execution.
