# SAP Sync

SAP sync is scheduled for 22:00 by default.

Use `.env` for all SAP credentials:

```env
SAP_ENABLED=false
SAP_SYNC_TIME=22:00
SAP_DB_HOST=
SAP_DB_PORT=
SAP_DB_NAME=
SAP_DB_USER=
SAP_DB_PASSWORD=
SAP_DB_TYPE=sqlserver
SAP_API_BASE_URL=
SAP_API_USERNAME=
SAP_API_PASSWORD=
```

No SAP password should be committed to GitHub.
